Find a Property - show me the map.

I used Node.js, Mongoose and Express to create REST API and then draw the data on google map. Markers on the map are connected to the cards, so hovering over card triggers animation to help localize the property. Front-end was done in Bootsrap.

Google Maps for everyone.

Google Maps cover 99% of the world and have 1 billion active users per month. The Maps JavaScript API lets you customize maps with your own content and imagery for display on web pages and mobile devices. The Maps JavaScript API features four basic map types (roadmap, satellite, hybrid, and terrain) which you can modify using layers and styles, controls and events, and various services and libraries.
