Hangman - Guess the secret word.

In this project, I built a hangman game. I used SVG to create the pole and the little figure. The objective of the game is to guess the random word, before whole hangman is visible. Event listeners let us only use keyboard letters. The database with secret words is fetched from MongoDB.

Scalable Vector Graphics

SVG stands for scalable vector graphics, and it is a file format that allows you to display vector images on your website. This means that you can scale an SVG image up and down as needed without losing any quality, making it a great choice for responsive web design.

Where SVGs differ is that the code inside the file can be copied and pasted directly on a page to display the image. This method allows for more control over manipulating the image.
