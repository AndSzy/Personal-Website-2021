Property Blog - fun with charts.

Main goal of this project was to present changes in property prices in Poland in approachable way. I created it using Bootstrap, which is guarantee of responsive, mobile-first design. The graphs are implemented in a way, that user can easly filter the data and maximize it at any given time. There are mortgage calculator and blog posts at the sidebar. Data are stored locally in JSON file, but it's possible to use API instead.

Simple data visualization.

Chart.js is a community maintained open-source library that helps you easily visualize data using JavaScript. It supports 8 different chart types (including bars, lines, & pies), and they’re all responsive. In other words, you set up your chart once, and Chart.js will do the heavy-lifting for you and make sure that it’s always legible.
